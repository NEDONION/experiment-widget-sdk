Experiment Widget SDK Demo

This page demonstrates the Experiment Widget SDK integration.

Note: The widget is loaded via the script tag below. Make sure your backend API is running at the specified endpoint.

How to Use

Simply add this script tag to any webpage:

<script
src="https://cdn.example.com/experiment-widget.js"
data-api-base="http://localhost:4000/api/v1"
data-experiment-id="exp_123"
data-user-key=""
async>
</script>

Parameters

  • data-api-base: Your backend API base URL
  • data-experiment-id: The experiment ID to run
  • data-user-key: Optional user identifier
