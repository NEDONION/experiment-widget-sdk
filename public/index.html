Experiment Widget SDK

Demo page

这个页面只是用来验证部署是否正常，同时预览浮窗。示例挂载在右下角（可用 data-position 调整）。

你可以直接复用下方的 script 标签，把 data-api-base 和 data-experiment-id 换成自己的：

<script src="./experiment-widget.js" data-api-base="https://api.example.com" data-experiment-id="demo" data-random-assignment="true" data-position="bottom-right"></script>
